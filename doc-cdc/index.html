Desconto Racional por Dentro.

In fact, since this script is meant to be run on the server, via nodejs (and not on the browser), it must export its functions to serve Express, which provides a robust set of features for web and mobile applications.

routes/routes~routes/routes.js

Summary.

CDC router.

Vercel seems to require a /api folder with an index.js file in it to work. Static files, such as .css and .html, lie in /public folder.

Therefore, it loads files directly from /public or /api, e.g.,

  • https://cdc-express.vercel.app/,
  • https://cdc-express.vercel.app/api/cgi?np=10&tax=0&pv=450&pp=500,
  • https://cdc-express.vercel.app/favicon.ico or
  • https://cdc-express.vercel.app/cd.css.

However, running locally, it would be

  • http://localhost:3000/api/cdc,
  • http://localhost:3000/api/cgi?np=10&tax=0&pv=450&pp=500,
  • http://localhost:3000/api/favicon.ico or
  • http://localhost:3000/api/cd.css.
